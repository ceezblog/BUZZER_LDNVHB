LED PANEL CONTROLLER for BUZZER LD NVHB

- Project originally created using VS2013 with traces of MFC although not using MFC
- Ported to VS2022 (tool set v143) with some modification in resouce.rc in order to get rid of MFC completely
- File session.txt is contain the data the app is using, any change in the app reflect in this file
- Name/text to display should be less than 9 chars due to limitation of the LCD panel
